# Lab Exercise 2 – Dynamic Styling

**Goal:** Change UI attributes programmatically based on user input.

### XML Task
Add two buttons: "Red" and "Blue", and one large `TextView` in the center.

### Activity Task
* When "Red" is clicked, change the `TextView` text to "STOP" and its text color to Red.
* When "Blue" is clicked, change the text to "GO" and the text color to Blue.

**Bonus:** Change the background color of the entire layout using `layout.setBackgroundColor()`.
