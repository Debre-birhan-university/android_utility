- creating a logo for the app we need to files
    - **Foreground Layer:** Your logo or brand symbol. It should be transparent around the edges.
    - **Background Layer:** A solid color or a simple pattern.


- steps to create an icon for our app
-  **Open your project** in Android Studio.
- In the Project window, right-click the **res** folder.
- Select **New > Image Asset**.
- **Configure the layers:**
    - **Icon Type:** Select "Launcher Icons (Adaptive and Legacy)".
    - **Foreground Layer:** Select your image file and use the **Resize** slider to fit it in the safe zone
    - **Background Layer:** Choose a color or an image.
- **Check Legacy:** Ensure the "Legacy" tab is configured so your app still looks good on older versions of Android.
- **Finish:** Click Next and then Finish. Android Studio will automatically generate all the different sizes ($hdpi$, $xhdpi$, $xxhdpi$, etc.) for you.
