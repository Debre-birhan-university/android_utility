### 1. Basic Setup
- create `textsViews.xml`. inside layout subfolder 
- clean the text and add these wrapper

- Add `android:orientation="vertical"` to your `LinearLayout` tag. This ensures every [[View]] you add stacks from top to bottom so they don't overlap.

### 2. Adding Widgets
Inside the `<LinearLayout>` tags, you will practice adding the following components:
- [[TextView]] - For labels and titles.
- [[Button]] - For user actions.
- [[EditText]] - For user input.
- [[ImageView]] - For visual content.
- [[CheckBox]] - For multiple-choice selections.
