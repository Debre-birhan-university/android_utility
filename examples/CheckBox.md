 Used when a user needs to select one or more options from a set. 
 ### Example Code 
 ```xml 
 <CheckBox 
 android:id="@+id/cb_agree" 
 android:layout_width="wrap_content" 
 android:layout_height="wrap_content" 
 android:text="I agree to the terms" 
 android:checked="true" />
 ```
 **`android:checked`**: Sets the default state (`true` or `false`).

## Attributes 
> [!NOTE] 
> 🔗 Every CheckBox inherits basic generic properties (like `id`, `layout_width`, `layout_height`, `background`, `visibility`, etc.) from the base [[View]] class, and text formatting properties (like `android:text`) from the [[TextView]] class.
